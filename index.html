Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Html defines the structure of a webpage.
  • The charset highlights the characters and symbols being used in the webpage.
  • The 'id' attribute is used to identify and select elements.
  • Img tag is used to insert image in a webpage and src tag defines the source of the image file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: to add all changes to the merge
  • git commit -m "add your message": to add message
  • git pull origin main: to pull the latest version of the code to your local machine
  • git push origin 'branch name': to sync changes to remote branch
